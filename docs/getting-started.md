# Inicio Rápido

Esta guía te ayudará a configurar DayliRegister y comenzar a gestionar tu negocio en minutos.

## Requisitos del Sistema

Antes de comenzar, asegúrate de tener:

- Un navegador actualizado (Chrome 90+, Firefox 88+, Safari 14+, Edge 90+)
- Conexión a internet para la primera carga
- Al menos 50MB de espacio disponible en tu dispositivo

## Primer Acceso

### 1. Abrir la Aplicación

Accede a DayliRegister desde tu navegador. La aplicación se cargará automáticamente.

### 2. Configuración Inicial

Al abrir la aplicación por primera vez, verás la pantalla principal con:

- Panel de navegación con 4 botones快捷:
  - **Reportes** - Acceder a estadísticas (próximamente disponible)
  - **Recepciones** - Gestionar recepciones de inventario
  - **Configuración** - Ajustes de la aplicación
  - **Tema** - Cambiar entre modo claro y oscuro
- Botón principal **"Nueva Sucursal"** para comenzar

### 3. Crear tu Primera Sucursal

Sigue estos pasos para agregar tu primera sucursal:

1. Toca el botón **"Nueva Sucursal"**
2. Completa los datos requeridos:
   - **Nombre** - Identificador de la sucursal
   - **Dirección** (opcional) - Ubicación física
   - **Teléfono** (opcional) - Contacto de la sucursal
3. Activa el toggle **"Activa"** para habilitar la sucursal
4. Toca **"Guardar Sucursal"** para crear

¡Listo! Tu primera sucursal está creada y lista para usar.

## Interfaz Principal

La pantalla principal de DayliRegister se divide en:

### Barra Superior

- 4 botones de acceso rápido:
  - Reportes (futuro)
  - Recepciones (futuro)
  - Configuración (futuro)
  - Cambio de tema

### Área de Acción

- Botón prominente para crear nuevas sucursales

### Lista de Sucursales

- Muestra todas las sucursales creadas
- Cada tarjeta muestra: nombre, dirección, teléfono y estado
- Acciones disponibles: editar y eliminar

## Solución de Problemas

### La página no carga

- Verifica tu conexión a internet
- Limpia la caché del navegador
- Intenta con otro navegador

### Los datos no se guardan

- Asegúrate de tener espacio en tu dispositivo
- Verifica que las cookies estén habilitadas
- La app guarda automáticamente en almacenamiento local

### Necesito ayuda adicional

Contacta al soporte técnico de DayliRegister con los detalles del problema.
